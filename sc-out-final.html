What is SC-OUT?

SC-OUT is an FPL (Fantasy Premier League) scouting dashboard that helps managers identify the best players to pick based on real-time data. It fetches live data from the official FPL API and uses a proprietary scoring algorithm to rank every player by their value, form, expected goal involvement, and differential potential. Player cards can be expanded to reveal advanced analytics powered by Claude AI.

How to Run Locally

SC-OUT is a self-contained HTML file — no build tools or server required.

  • Clone or download the repository from GitHub
  • Open index.html in any modern browser (Chrome, Firefox, Edge)
  • The app auto-fetches live FPL data on load via the AllOrigins CORS proxy
  • No environment variables or local server needed for core functionality

How to Deploy

This app can be deployed to any static hosting provider:

  • Web01 / Web02: Copy index.html into the web root (/var/www/html/) on both servers using scp or rsync
  • Load Balancer (Lb01): Configure Nginx or HAProxy to round-robin between Web01 and Web02. Both servers must serve the same file to ensure consistent user experience
  • Verify deployment by accessing the app via the load balancer IP/domain and refreshing several times to confirm traffic distribution

External APIs Used

🏴󠁧󠁢󠁥󠁮󠁧󠁿 FPL Bootstrap API 🔀 corsproxy.io (primary) 🔀 AllOrigins (fallback) ✦ Claude AI (Anthropic)
  • FPL Bootstrap Static API — https://fantasy.premierleague.com/api/bootstrap-static/ — Official Premier League API providing live player stats, team data, and gameweek information. Public API; no key required.
  • corsproxy.io / AllOrigins / CodeTabs — A chain of public CORS proxies tried in sequence. If the primary proxy fails (e.g. a 520 error), the app automatically falls back to the next. No API key required.
  • Claude AI (claude-sonnet-4-20250514) — Anthropic's API powering the AI player analysis text. API key is injected server-side by the platform and is never exposed in the client-side code or repository.

SC-OUT Scoring Algorithm

Each player receives a composite SC-OUT Score calculated from: Points-per-million (value), current form, expected goal involvement (xG + xA), minutes reliability, differential ownership bonus, bonus points, and clean sheet contributions. Scores range from ~0 (Avoid) to 100+ (Elite).

Challenges & Solutions

  • CORS Restrictions: The FPL API does not allow direct browser requests. Solved using a chain of 3 CORS proxies (corsproxy.io → AllOrigins → CodeTabs) with automatic fallback if one fails.
  • API Downtime: Implemented graceful error handling with retry functionality and user-friendly error messages.
  • AI Latency: Claude AI analysis is generated on-demand per card expansion to avoid blocking the initial data load.
